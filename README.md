## Getting Started

### Installation

#### 1. Clone Repository

```bash
git clone https://github.com/natebabyak/portfolio.git
cd portfolio
```

#### 2. Install Dependencies

```bash
bun install
```

#### 3. Run Local Development Server

```bash
bun run dev
```

### Customization

Although this portfolio is not designed to be a reusable template, the structure is simple to understand and modify.
